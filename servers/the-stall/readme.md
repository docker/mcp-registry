MCP Endpoint: https://the-stall.intuitek.ai/mcp

The Stall is a remote MCP server providing 173 pay-per-call financial data capabilities via the x402 micropayment protocol. Agents pay per call in USDC on Base mainnet — no API keys, no accounts, no monthly fees.

**Capabilities include:** US stocks (prices, fundamentals, earnings, analyst ratings, insider trades, congressional trades), crypto (prices, DeFi analytics, token security, whale tracking), macro (Fed rates, Treasury yields, economic calendar, CPI), options chains, prediction markets, OSINT, and more.
