# APIsec NG MCP Server

MCP server for interacting with the APIsec NG platform. Manage applications, initiate security scans, and review scan results.

## Documentation

For detailed documentation, see:
- [GitHub Repository](https://github.com/kauaim/apisec-ng-mcp-server)
- [APIsec NG Documentation](https://docs.apisecapps.com)

## Features

- **Application Management**: Create applications from OpenAPI, GraphQL, or Postman collections
- **Security Scanning**: Initiate and monitor security scans
- **Detection & Reporting**: Get security findings and OWASP coverage reports
- **API Integration**: Full integration with APIsec NG platform

## Configuration

The server requires an APIsec NG Personal Access Token (PAT) for authentication.

See the main [README.md](https://github.com/kauaim/apisec-ng-mcp-server/blob/main/README.md) for configuration details.
