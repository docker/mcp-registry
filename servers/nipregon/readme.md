# NIPRegon MCP Server

Polish company registry lookup for AI assistants: 4.4M businesses from official public registers (KRS, REGON, CEIDG, Ministry of Finance VAT white list, financial statements). Tools: company search, full registry data by NIP, yearly financial statements, and VAT white-list checks before payments. Read-only, official public data only.

- Repository: https://github.com/AMR-DEV-PS/nipregon-mcp
- Documentation: https://nipregon.pl/mcp
- Official MCP registry: `pl.nipregon/nipregon`

Remote streamable HTTP endpoint: `https://api.nipregon.pl/mcp` (rate-limited, optional API key).
