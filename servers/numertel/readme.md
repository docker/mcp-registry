# NumerTel MCP Server

Polish phone number lookup for AI assistants: who called, spam and scam checks, the state UKE DNO anti-spoofing registry, verified official-hotline whitelist, and daily phone-phishing indicators from CERT Polska data. Backed by [NumerTel.pl](https://numertel.pl) (130M numbers). Read-only, no personal data returned.

- Repository: https://github.com/AMR-DEV-PS/numertel-mcp
- Developer docs: https://numertel.pl/dla-deweloperow
- Official MCP registry: `pl.numertel/numertel`

Remote streamable HTTP endpoint: `https://numertel.pl/api/mcp` (rate-limited, optional API key).
