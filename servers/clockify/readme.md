# Clockify MCP Server

This entry registers the Clockify MCP server for time tracking in the Docker MCP Registry.

- Project: https://github.com/skafaliev-dev/clockify-mcp-server
- Documentation: https://github.com/skafaliev-dev/clockify-mcp-server#readme

## Configuration

- `CLOCKIFY_API_KEY`: Your Clockify API key
- `CLOCKIFY_API_BASE_URL`: Optional, defaults to https://api.clockify.me/api/v1

See the main project README for usage and setup instructions.
