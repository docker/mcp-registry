# Aspose Words Cloud MCP Server

A Model Context Protocol server that exposes [Aspose.Words Cloud](https://products.aspose.cloud/words/) endpoints as tools for AI agents. Documents are transmitted as base64 strings within JSON-RPC messages — no shared volumes required.

## Tools

- **convert_document** — Convert documents between formats (DOCX, PDF, HTML, etc.)
- **split_document** — Split a document into multiple files by page or range
- **mail_merge** — Perform mail merge operations on a document template

## Setup

1. Create a free Aspose app at [dashboard.aspose.cloud/applications](https://dashboard.aspose.cloud/applications) and obtain your `Client ID` and `Client Secret`.
2. Set `ASPOSE_CLIENT_ID` and `ASPOSE_CLIENT_SECRET` environment variables.
