# Locize MCP Server

Official remote MCP server for [Locize](https://www.locize.com), the translation management system from the i18next maintainers.

- Endpoint: `https://mcp.locize.app` (streamable HTTP)
- Auth: OAuth 2.0 (RFC 9728 discovery) or a Locize personal access token (`lz_pat_...`) as a Bearer token
- 26 tools: translations read/write, translation memory search, glossary, styleguide, screenshot context, publish versions, branches, languages, namespaces
- Documentation: https://www.locize.com/docs/integration/mcp
