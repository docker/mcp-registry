Ssemble AI Clipping MCP server lets you create short-form video clips from YouTube videos using any AI assistant.

## Features

- Create AI-generated short clips with captions, background music, gameplay overlays, and meme hooks
- Browse 9+ caption templates, 45+ music tracks, 30+ gameplay videos, and 20+ meme hooks
- Each clip gets a viral score, AI-generated title, and description
- Supports multiple clip lengths (30s to 10min)
