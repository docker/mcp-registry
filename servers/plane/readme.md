# Plane MCP Server

A Model Context Protocol (MCP) server for [Plane](https://plane.so) - the open-source project management platform.

## Documentation

- [Plane MCP Server Documentation](https://github.com/makeplane/plane-mcp-server)
- [Plane Documentation](https://docs.plane.so/)

## Features

- **55+ Tools** for comprehensive Plane integration
- **Project Management**: Create, update, and manage projects
- **Work Items**: Full CRUD operations on issues/tasks
- **Cycles & Modules**: Sprint and module management
- **Initiatives**: Strategic initiative tracking
- **Intake**: Manage incoming work items

## Getting Started

1. Generate an API key from your Plane workspace at `/settings/account/api-tokens/`
2. Configure the server with your workspace slug and API key
3. Optionally set a custom base URL for self-hosted Plane instances
