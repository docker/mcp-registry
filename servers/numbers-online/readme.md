# Numbers Online

Phone-number trust & intelligence as a supplementary signal for AI agents: line
type, range carrier, country, CNAM caller name, STIR/SHAKEN verstat, a labeled
spam-risk signal, and a first-party do-not-contact (DNC) signal — each returned
as a supplementary, low-confidence signal, with an Ed25519-signed receipt.

- Documentation: https://numbers.online/docs
- Homepage: https://numbers.online

Public discovery methods (`initialize`, `ping`, `tools/list`) are keyless. Tool
calls require a free API key sent as `Authorization: Bearer nol_…`; get one via
self-service signup at https://numbers.online.
