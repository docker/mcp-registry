# Aspose BarCode Cloud MCP Server

An MCP server that enables AI assistants to generate, recognize, and scan barcodes using the [Aspose BarCode Cloud API](https://products.aspose.cloud/barcode/). Supports 60+ barcode symbologies including QR, Code128, DataMatrix, EAN, PDF417, Aztec, and more.

## Tools

- **generate_barcode** - Create barcode images from text data
- **recognize_barcode** - Identify specific barcode types in images
- **scan_barcode** - Auto-detect and read barcodes from images
- **list_barcode_types** - List all supported barcode symbologies

## Setup

1. Get free API credentials at [dashboard.aspose.cloud](https://dashboard.aspose.cloud/applications) (150 free requests/month)
2. Configure your `ASPOSE_CLOUD_CLIENT_ID` and `ASPOSE_CLOUD_CLIENT_SECRET`
3. Set a local directory for barcode file exchange
