The first Digital Out-of-Home (DOOH) advertising MCP server. Enables AI agents to discover, query, and transact with 5,000+ digital screens across the US.

## Features

- **Inventory Discovery** — Find screens by location, venue type, audience demographics
- **Real-Time Audience Signals** — Edge AI powered (face detection, gaze tracking, emotion, audio classification)
- **Campaign Management** — Create, monitor, and optimize DOOH campaigns
- **Ad Context Protocol (AdCP v3)** — Full compliance for agentic advertising workflows
- **Usage-Based Billing** — Free tier included, pay-per-use via Stripe
- **Proof of Play** — Ed25519 cryptographic verification of ad delivery

## Getting Started

No API key is required to get started. Use the `register_partner` tool to create a free account and receive an API key for advanced features.
