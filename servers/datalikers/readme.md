# DataLikers MCP Server

Access Instagram data through DataLikers API.

## Documentation

https://datalikers.com/mcp

## Features

- Search users by demographics (age, gender, race, location)
- Get user profiles, posts, comments, stories, highlights
- Search hashtags and locations
- Run custom SQL queries on Instagram data
- Face analysis data (age, gender, race, emotion)
- 28 tools available

## Getting Access Key

1. Register at https://datalikers.com
2. Go to Tokens page
3. Create a new token
