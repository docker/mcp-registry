## Tools (8)

- **search_agents** — Search for agent-ready websites, APIs, and services ranked by agentic readiness score
- **get_site_details** — Get the full agentic readiness report for a specific domain
- **get_stats** — Get current statistics for the index
- **list_categories** — List all categories with site counts and average scores
- **get_top_sites** — Get the highest-scored agent-ready sites, optionally by category
- **submit_site** — Submit a URL for crawling and scoring
- **verify_mcp** — Probe any URL to verify it is a live MCP server
- **register_monitor** — Monitor a domain's agentic readiness score for regressions
