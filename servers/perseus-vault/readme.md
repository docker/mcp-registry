# Perseus Vault

Persistent, encrypted memory for AI agents — hybrid semantic + keyword recall,
corrections, and decision journaling. One Rust binary, MIT licensed.

- Source: https://github.com/Perseus-Computing-LLC/perseus-vault
- Docs: https://github.com/Perseus-Computing-LLC/perseus-vault#readme

Requires a bearer token for the hosted endpoint (`PERSEUS_VAULT_MCP_TOKEN`).
Self-hosting is also supported — point the remote URL at your own instance.
