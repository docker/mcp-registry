# GammaRips Options Intelligence

Documentation: https://gammarips.com/developers

Anti-firehose options-flow intelligence for AI agents: a hard-curated daily
candidate pool, point-in-time features, realized MFE/MAE opportunity surfaces,
outcome labels, and methodology playbooks. There is deliberately no "pick"
endpoint — the agent reasons over data primitives to its own conclusion.

Free anon tier works without authentication (daily report, playbooks,
explainers). Pro tools require a subscription key sent as
`Authorization: Bearer gr_live_...` — see https://gammarips.com/pricing.

Source: https://github.com/DevDizzle/gammarips-mcp (MIT).
Educational data, not investment advice.
