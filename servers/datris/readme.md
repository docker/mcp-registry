# Datris

AI Agent-Native Data Platform — ingest, validate, transform, and query data from enterprise sources.

## Features

- **35 MCP tools** across 7 categories: Pipeline Management, Vector Search, Database Queries, Metadata Discovery, AI, Configuration, and Managed Service
- **AI-powered data quality** — generate Python validation scripts from plain-English instructions
- **AI-powered transformations** — generate Python transformation scripts from natural language
- **Vector search** — semantic search across Qdrant, Weaviate, Milvus, Chroma, and pgvector
- **RAG workflows** — retrieve document chunks and pass to AI for synthesized answers
- **Multi-destination** — route data to PostgreSQL, MongoDB, Kafka, and vector stores
- **Secrets management** — HashiCorp Vault integration for secure credential storage

## Getting Started

1. Sign up for a free 14-day trial using the `signup_trial` tool (no API key needed)
2. Receive your API key and MCP endpoint URL
3. Reconnect with your API key to access all 35 tools
4. Create a pipeline, upload data, and start querying

Or sign up at [datris.ai](https://datris.ai) and configure your MCP client with the API key.
