# Cabin Flights MCP Server

AI-first travel booking with natural language and crypto micropayments.

## Features

- **Natural Language Flight Search**: Search flights using conversational queries
- **Crypto Micropayments**: Pay-per-use with USDC on Base/Ethereum networks
- **Multi-Network Support**: Base, Ethereum, Polygon blockchain networks
- **Group Booking Intelligence**: Handle complex group travel scenarios
- **Real-time Data**: Live flight prices and availability via Duffel/Amadeus APIs
