# Cabin Flights MCP Server

AI-first travel booking with natural language. Search and book real flights with crypto micropayments (x402).

## Documentation

Full documentation is available at: https://github.com/yolo-maxi/cabin/tree/main/packages/agent

## Features

- Natural language flight search
- Real airline API integration  
- x402 crypto micropayments ($0.01 USDC per search)
- Multi-network support (Base, Ethereum, Polygon)
- Flight booking status tracking

## Configuration

Requires an EVM wallet private key for x402 payments. See the [setup guide](https://github.com/yolo-maxi/cabin/blob/main/packages/agent/README.md) for details.
