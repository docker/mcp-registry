# Nex MCP Server

Organizational context and memory for AI agents. Connect email, CRM, Slack, meetings, and 100+ tools into one knowledge graph.

## Documentation

- [GitHub Repository](https://github.com/nex-crm/nex-as-a-skill)
- [Website](https://nex.ai)
- [npm Package](https://www.npmjs.com/package/@nex-ai/nex)

## Features

- **Knowledge Graph**: Automatically extracts entities, relationships, and insights from unstructured text
- **60 MCP Tools**: Full CRUD for records, objects, relationships, lists, tasks, notes, and notifications
- **AI-Powered Search**: Natural language queries across your entire organizational context
- **Integrations**: Connect Google Calendar, Outlook, Slack, and 100+ tools via OAuth
- **Meeting Bot**: Joins calls on any platform and feeds transcripts into the context graph
- **Proactive Notifications**: AI-driven alerts for deals, relationships, and context changes

## Configuration

Set the \`NEX_API_KEY\` environment variable, or use the \`register\` tool to create an account.
