# Scalix World MCP Server

Documentation: https://docs.scalix.world/mcp

The hosted MCP server for Scalix World, the AI-native neocloud. Discovery
(`initialize`, `tools/list`) is unauthenticated; `tools/call` requires a Scalix
World API key as a Bearer token (RFC 6750 challenge on 401). Get a free key at
https://scalix.world (no card required). Also listed in the official MCP
registry as `world.scalix/cloud`.
