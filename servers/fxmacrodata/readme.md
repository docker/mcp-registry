# FXMacroData

FXMacroData provides a hosted Model Context Protocol server for official-source
FX, macroeconomic, and central-bank data.

Documentation:

- https://fxmacrodata.com/documentation/mcp-server
- https://fxmacrodata.com/documentation/reference
- https://fxmacrodata.com

Server URL:

- https://mcp.fxmacrodata.com/mcp

The server supports public USD data without authentication. Full multi-currency
access is available through FXMacroData account credentials.
