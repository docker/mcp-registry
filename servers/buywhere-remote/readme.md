# BuyWhere MCP Server

Real-time product search and price comparison for Southeast Asia.

## Features

- Search 260K+ products from Lazada, Shopee, FairPrice, Courts, and other major SEA merchants
- Compare prices across multiple retailers in real-time
- Check product availability by store
- Designed for Singapore and Southeast Asia markets

## Getting Started

1. Get a free API key at [buywhere.ai/api-keys](https://buywhere.ai/api-keys)
2. Connect via the remote MCP endpoint: `https://api.buywhere.ai/mcp`
