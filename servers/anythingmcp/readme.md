# AnythingMCP

Self-hostable gateway that turns any REST/GraphQL API, SQL/NoSQL database or existing MCP server into no-code connectors for AI assistants (Claude, ChatGPT, Gemini, Copilot, Cursor), exposed over the Model Context Protocol.

- Website: https://anythingmcp.com
- Documentation: https://anythingmcp.com/guides
- Source code: https://github.com/HelpCode-ai/anythingmcp

The listed remote endpoint (`https://cloud.anythingmcp.com/mcp/demo`) is a public, read-only demo that describes the product and how to use it. It requires no authentication and exposes no customer data.
