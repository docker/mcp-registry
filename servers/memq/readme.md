Docs: https://multinex.ai/memq/

Get a MemQ API key: https://billing.multinex.ai/signup?product=memq&utm_source=docker-mcp&utm_medium=registry&utm_campaign=memq-install

After signup or checkout, create an API key from the Multinex Billing dashboard and enter it as `MEMQ_API_KEY` in Docker Desktop MCP Toolkit. The Docker MCP entry sends it as `Authorization: Bearer <MEMQ_API_KEY>` to the hosted MemQ endpoint at `https://mcp.multinex.ai/mcp/v1`.
